# csun-campus-guesser

Comp 484 - Web Engineering - Lab 4 - CSUN Google Maps API Game

## Description

CSUN Campus Guesser is a web-based game that challenges your knowledge of the California State University, Northridge (CSUN) campus. Built with the Google Maps JavaScript API, this GeoGuesser-style game presents you with names of various campus locations, and your task is to click on the correct spot on the map. It's a fun way to see how well you really know your campus!

## How to Run

To run this project locally, you will need a Google Maps API key.

1. Clone this repository to your local machine.
2. Obtain a Google Maps JavaScript API key from the [Google Cloud Console](https://console.cloud.google.com/). Make sure the "Maps JavaScript API" is enabled for your project.
3. Open the `index.html` file and replace the placeholder API key in the script tag with your own key.
4. Open `index.html` in your favorite web browser. For best results, it's recommended to use a local web server (like the VS Code Live Server extension).
